Save Instructions

Due to the primitive nature of IE8, this file must be saved manually. To do so, simply follow these steps. (() indicates windows specific instuctions)
  1. Click anywhere below the line
  2. Use the Select All shortcut (ctrl+A)
  3. Use the Copy shortcut (ctrl+C)
  4. Create a new text document (right click desktop or explorer -> new text document)
  5. Click the document and paste to it (ctrl+v)
  6. Save the document with the extension .html instead of .txt

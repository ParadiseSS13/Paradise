Space Station 13 - Modules - Types — Var Details

security
/obj/machinery/computer/security

The root type of all camera consoles. When used, these open a UI that has a left-hand sidebar displaying a list of active cameras in the console's assigned camera network. The bulk of the window on the right hand side displays a camera feed of the selected camera. This feed will attempt to refresh every time process() is called by SSobj as long as there are active watchers, living or dead.
